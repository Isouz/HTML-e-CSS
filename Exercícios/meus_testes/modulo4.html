Aula 1 – O iframe ainda pode ser usado?

A palavra iFrame é uma abreviação usada para se referir a “quadro embutido”, uma tag HTML usada para incorporar o conteúdo de uma página em outra. Vamos entender melhor o que é iFrame? A princípio, cada site consiste em um único documento HTML. Caso queira inserir outros elementos, é necessário usar um frame. Isso torna possível integrar documentos, imagens ou vídeos adicionais no documento HTML

Por exemplo, abaixo temos um iframe que nosmostra a pagina do curso em video.

é quase como se houvesse uma nova guia dentro da pagina.

Vale ressaltar que alguns sites possuem proteção contra iframe e não conseguimos usar esse metodo com eles. É o caso do site do Google. Veja o exemplo abaixo:

para criar um iframe basta usar a tag <iframe> </iframe>

e dentro do src colocamos o link do conteúdo.
<iframe src="https://www.cursoemvideo.com" frameborder="0"> </iframe>

Aula 2 – Configurando iframes.

Alguns navegadores de celular não possuem suporte para iframe. Então, tudo que colocarmos entre as tags vai aparecer para o usuario apenas se o dispositivo não tiver suporte para iframe. Nesse caso, podemos colocar uma mensagem de erro:

<iframe src="https://www.cursoemvideo.com" frameborder="0"> [Seu navegador não é compativel] </iframe>

Ou melhor, podemos simplesmente colocar o link.

<iframe src="https://www.cursoemvideo.com" frameborder="0"> <a href="https://www.cursoemvideo.com">CursoemVideo</a> </iframe>

Tamanhos.

Também podemos alterar o tamanho do iframe. Por padrão ele possui 300 x 150 (px), mas podemos adicionar um paramentro height e width dentro da tag:

<iframe src="https://www.cursoemvideo.com" frameborder="0" height="500" widht="500"> </iframe>

Também podemos redimensiona-las com o uso do CSS. Caso exista a configuração na CSS e na tag, sempre prevalecerá a CSS.

Parametro scrolling.

Esse árametro serve para habilitar ou desabilitar a barra de rolagem no elemento iframe. O atributo possui os seguintes valores:
  • auto: é o valor padrão, se o conteúdo exceder o tamanho do iframe ele cria uma barra de relagem.
  • yes: serve para exibir a barra de rolagem;
  • no: serve para esconder a barra de rolagem;
  • auto: a barra de rolagem é exibida conforme a necessidade do elemento.
<iframe src="https://www.cursoemvideo.com" frameborder="0" scrolling=""> </iframe>

frameborder.

Esse atributo é utilizado para modificar a borda padrão do elemento. Ele só aceita duas possibilidades como valor: 0 e 1. 0 ele tira a borda e 1 ele coloca a bordade 1px em volta do iframe (podemos personaliza-la na CSS usando o border).
